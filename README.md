# UPSC Hub 🚀

A full-stack web application built using React and Flask.

## 🔧 Tech Stack
- Frontend: React
- Backend: Flask
- Language: JavaScript, Python

## ✨ Features
- User Signup/Login
- API Integration
- Full-stack architecture

## 📁 Project Structure
frontend/ → React app  
backend/ → Flask server  

## 🚀 How to Run

### Frontend
cd frontend  
npm install  
npm start  

### Backend
cd backend  
pip install -r requirements.txt  
python app.py
